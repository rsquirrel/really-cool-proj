1. To build the compiler and interpreter, cd into 'src' directory and type 'make'. Two files 'tmlc' and 'tml.jar' will appear in the same directory.

2. To compile an TML source file, type 'tmlc xxx.tml'. A file named 'xxx.tmb' will appear in the same directory of the .tml file.

3. To execute the .tmb file, type 'tml xxx.tmb'.
